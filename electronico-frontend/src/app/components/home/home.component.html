escripción del Proyecto

Crear una aplicación de gestión de pedidos para un sistema de comercio electrónico. La aplicación debe permitir crear, leer, actualizar y eliminar pedidos, así como listar todos los pedidos y obtener detalles de un pedido específico. Un pedido tiene un cliente, una lista de productos y un total. Los productos y clientes deben almacenarse en bases de datos relacionales, mientras que los pedidos deben almacenarse en una base de datos no relacional.
